### Adding a website to your bookmarks. 
This can be accomplished by first browsing to the desired website then pressing the tile menu button. Then from the selecting the bookmarks tile option. Then selecting the bottom option (bookmark symbol with plus sign) in order to add the website you were at (should still show in the bottom url bar) to your favorites. 
 
### Changing Home Page
To change the homepage, first navigate to the desired webpage. From there, go into the main menu by pressing the tile button. From the four main options, select the settings tile. Next, select the “Set Homepage” tile to set the webpage currently being browsed as the homepage.
 
### Changing Text Size
To change text size of the browser, first go into the main menu via the tile button. From the four main options, select the settings tile. Next, select the text size tile, which brings up a menu of various font sizes. The user would select the desired size, after which the indicator shows changes have been made.
